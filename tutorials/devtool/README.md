# 开发工具

## 日志工具

### Log

Android 的日志工具类（`android.util.Log`）提供一下方法

- Log.v：verbose
- Log.d：debug
- Log.i：info
- Log.w：warn
- Log.e：error


### Logcat

Logcat 是 Android Studio 提供的开发工具，具有以下功能：

- 快捷输入

    - logv
    - logd
    - logi
    - logw
    - loge
    - logt

- 日志查询

    设备、包名、日志级别、关键字、正则匹配
